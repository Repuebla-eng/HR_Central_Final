# **App Name**: HR Central

## Core Features:

- Employee Profile Management: Create, read, update, and delete employee profiles with detailed information, including personal data, job title, department, and status.
- Job Title and Competency Definition: Define job titles and their required competencies (as per ISO 9001), including competency names, descriptions, required levels, and evidence needed.
- Competency Assessment: Assess employee competencies against job title requirements, identifying gaps and recording assessment dates and assessor IDs.
- Training Plan Management: Create and manage training plans to address competency gaps, assigning courses and setting deadlines.
- Role-Based Access Control: Implement a role-based system that relies on custom claims to determine user access permissions
- Authentication and Authorization: Implement secure authentication and authorization flows using Firebase Authentication to manage user access.
- AI-Powered Competency Gap Analyzer: Integrate Genkit to create a tool that suggests personalized training plans based on detected skill gaps.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to convey trust and professionalism.
- Background color: Light Gray (#ECEFF1) for a clean, modern interface.
- Accent color: Teal (#009688) for call-to-action elements and highlights.
- Body font: 'Inter', a sans-serif font, known for a neutral and readable aesthetic.
- Headline font: 'Space Grotesk', a sans-serif font to maintain consistency and simplicity in headings and titles.
- Use consistent and professional icons from a library like Feather Icons for navigation and actions.
- Maintain a clean and well-spaced layout using a grid system to ensure consistency across all pages.
